************************************************************
FindFace Android demo app build guide
************************************************************


The FindFace open-source Android demo app allows you to explore wide possibilities of our `facial recognition solutions <http://findface.pro>`__ and learn by example how you can integrate them into your own applications.

Here you can find a step by step guide to building the FindFace Android demo app from source code (provided as an Android project).
